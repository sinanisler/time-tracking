Advanced time tracking plugin with drag-to-select calendar, category management, and detailed time logging for WordPress users.

== Description ==

Time Tracking is a powerful and intuitive time management plugin for WordPress that helps users track their work hours, manage tasks, and monitor productivity. Built with modern web technologies, it provides a seamless calendar-based interface for efficient time tracking.

= Key Features =

* **Interactive Calendar** - Full-featured calendar with drag-to-select time blocks
* **Task Management** - Create, edit, and organize tasks with detailed descriptions
* **Category System** - Organize tasks with custom categories and color coding
* **Time Logging** - Built-in timer to track actual time spent on tasks
* **User Privacy** - Each user has their own private calendar and data
* **Customizable Settings** - Configure working hours, days, and timezone
* **Data Management** - Export and import your time tracking data
* **Dark Mode Support** - Choose between light, dark, or automatic color schemes
* **Role Management** - Control which user roles can access the plugin
* **Translation Ready** - Full support for internationalization

= Perfect For =

* Freelancers tracking billable hours
* Teams managing project time
* Individuals monitoring productivity
* Remote workers logging work hours
* Agencies tracking client projects

= User-Friendly Interface =

The plugin features a modern, responsive interface built with:

= Privacy & Security =

* Each user's data is completely private and isolated
* Only users with appropriate roles can access the plugin

= Customization Options =

* Set your timezone or use WordPress default
* Configure working hours (start and end time)
* Select working days (customize which days are shown)
* Choose color scheme (light, dark, or auto)
* Manage user role access

== Installation ==

1. Upload the `time-tracking` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Time Tracking' in the WordPress admin menu
4. Configure your settings in 'Time Tracking > Settings'
5. Start tracking your time!

= Minimum Requirements =

* WordPress 5.8 or greater
* PHP version 8.0 or greater
* MySQL version 5.6 or greater

== Frequently Asked Questions ==

= Can multiple users use this plugin? =

Yes! Each user has their own private calendar and data. Users cannot see or access other users' time tracking information.

= What user roles can access the plugin? =

By default, administrators, editors, authors, and contributors can access the plugin. You can customize this in the Settings page.

= Can I export my data? =

Yes! You can export all your time tracking data as a JSON file from the Settings page. This includes all tasks, categories, and time logs.

= Does this plugin work with block themes? =

Yes, the plugin is compatible with both classic and block themes.

= Is the plugin translation ready? =

Yes, the plugin is fully internationalized and ready for translation. Translation files can be added to the /languages directory.

= Can I customize the working hours? =

Yes, you can set custom start and end times for your workday in the Settings page.

= Does it support dark mode? =

Yes! You can choose between light mode, dark mode, or automatic (based on system preference) in the Settings page.
