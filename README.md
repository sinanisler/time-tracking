# Time Tracking

Advanced time tracking plugin with drag-to-select calendar, category management, and detailed time logging for WordPress users.

## Description

Time Tracking is a powerful and intuitive time management plugin for WordPress that helps users track their work hours, manage tasks, and monitor productivity.

## Features

- Interactive calendar with drag-to-select time blocks
- Task management with detailed descriptions
- Category system with color coding
- Built-in timer for tracking time
- User privacy - each user has their own private data
- Customizable settings (working hours, timezone, dark mode)
- Data export/import functionality
- Translation ready

## Installation

1. Upload the `time-tracking` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Time Tracking' in the WordPress admin menu
4. Configure your settings and start tracking!
